## Endpoints

### Auth
```
POST - /api/auth/signup
POST - /api/auth/signin
```

### Products
```
GET - /api/products
GET - /api/products/:id
POST - /api/products
PUT - /api/products/:id
DELETE - /api/products/:id
```

### Users
```
POST - /api/users
```
